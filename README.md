# WheelShare 🚗

A P2P vehicle rental platform where users can rent or list bikes and cars.

## Tech Stack
- **Frontend:** React 19 + Vite + Tailwind CSS v4
- **Backend:** Node.js + Express
- **Database:** MongoDB (Mongoose)
- **Auth:** JWT + bcryptjs

## Getting Started

**Prerequisites:** Node.js v18+, MongoDB running on port 27017

```bash
# Backend
cd wheelshare-server
node index.js

# Frontend (new terminal)
cd wheelshare-client
npm run dev
```

App runs at `http://localhost:5173` — API at `http://localhost:5000`
